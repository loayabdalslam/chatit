🤖 Chatbot Widget Test

Welcome to the chatbot widget testing environment! This page simulates a real website where your chatbot widget will be embedded.

🧪 How to Test

  1. Look for the chat button in the bottom-right corner
  2. Click the button to open the chat window
  3. Send a test message and wait for the response
  4. Try different message types to test the AI responses
  5. Test the widget on different screen sizes

✨ Widget Features

  • Custom Design: Fully customizable colors, fonts, and animations
  • Real-time Chat: Instant messaging with AI responses
  • Responsive: Works perfectly on desktop, tablet, and mobile
  • Secure: API key authentication for secure communication
  • Error Handling: Graceful error messages and connection recovery
  • Easy Integration: Simple copy-paste installation
