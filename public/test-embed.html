🚀 API Integration

Use these endpoints to integrate the chatbot into any system:

GET /api/chatbot/{chatbotId} - Get chatbot configuration
POST /api/chat - Send message and get response
GET /api/embed/{chatbotId} - Get embed code for any website

📋 Embed Code Example

<script>
(function() {
  var script = document.createElement('script');
  script.src = 'https://your-domain.com/widget.js';
  script.setAttribute('data-chatbot-id', 'your-chatbot-id');
  script.setAttribute('data-api-url', 'https://your-domain.com');
  document.head.appendChild(script);
})();
</script>
