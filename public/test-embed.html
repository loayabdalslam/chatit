ChatBase Widget Integration Tests

This page demonstrates different ways to integrate the ChatBase widget into your website using the new iframe-based approach that eliminates CORS issues.

1. Inline Widget (Embedded in Container)

Widget embedded directly in a page container:

<script src="https://fast-bulldog-662.convex.app/embed.js"></script> <div id="my-chatbot"></div> <script> ChatBaseWidget.embed({ chatbotId: 'test-bot-123', containerId: 'my-chatbot', width: '400px', height: '600px', position: 'inline' }); </script>

2. Auto-embed with Data Attributes

Simply add data attributes to any div:

<script src="https://fast-bulldog-662.convex.app/embed.js"></script> <div data-chatbase-widget data-chatbot-id="test-bot-123" data-position="inline" data-width="400px" data-height="600px"> </div>

3. Floating Widget (Bottom Right/Left)

Add a floating chat button to your website:

<script src="https://fast-bulldog-662.convex.app/embed.js"></script> <script> ChatBaseWidget.embed({ chatbotId: 'test-bot-123', position: 'bottom-right', // or 'bottom-left' width: '400px', height: '600px' }); </script>

4. Direct iframe (No JavaScript Required)

For the simplest integration, use a direct iframe:

<iframe src="https://fast-bulldog-662.convex.app/widget.html?chatbotId=test-bot-123" width="400" height="600" style="border: none; border-radius: 12px; box-shadow: 0 4px 20px rgba(0,0,0,0.1);"> </iframe>
